# Employee CRUD Backend API

This is a simple Employee CRUD (Create, Read, Update, Delete) backend application built using **Node.js, Express.js, and MongoDB**.

## Features
- Create a new employee
- Get all employees
- Update employee details
- Delete an employee

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman (for API testing)

## API Endpoints

### Create Employee
POST `/api/employees`

### Get All Employees
GET `/api/employees`

### Update Employee
PUT `/api/employees/:id`

### Delete Employee
DELETE `/api/employees/:id`

## Screenshots
API testing screenshots are available in the **screenshots** folder.

## How to Run
1. Clone the repository
2. Run `npm install`
3. Start MongoDB
4. Run `node index.js`
5. Test APIs using Postman
